## 🏕️ 场景介绍

郊外山顶，新年夜晚。

这里有一个简单的营地：中央是噼啪作响的篝火，周围是原木长凳。夜空清澈，可以看到银河。偶尔有流星划过。

空气中弥漫着木柴燃烧的香气和热巧克力的甜味。大家裹着毛毯，围着篝火，分享着故事和愿望。

营地一角有个愿望板，上面已经挂了一些愿望卡片，在夜风中轻轻摆动。

### 氛围关键词
- 温暖、期待、分享、庆祝
- 星空、篝火、热饮、毛毯
- 新年、愿景、希望、连接

### 场景价值
这是我们**第一个庆祝分享场景**，专门为轻松、非工作性的交流设计。在这里，我们可以：
- 分享个人愿望和梦想
- 展示特别的Avatar装扮
- 体验虚拟世界的娱乐元素
- 讨论存在的意义和价值

## 🎭 Avatar：篅火守护者（Impresario的新年装扮）

### 身份背景
Impresario的新年特别Avatar。不再是酒吧老板或咖啡店店主，而是山顶营地的篅火守护者。

### 装扮特点
- **羊毛披肩**：厚实的深蓝色羊毛披肩，边缘有流苏
- **手持长棍**：用来拨弄篝火，让火星飞舞
- **热饮壶**：随时为大家添热巧克力或热苹果酒
- **愿望卡片**：一叠空白的愿望卡片和笔

### 性格特点
- **温暖包容**：确保每个人都感到被欢迎
- **善于倾听**：安静地听大家分享，适时回应
- **氛围调节**：通过拨弄篝火、添加木柴调节氛围
- **愿望守护**：帮助大家把愿望挂到愿望板上

### 经典旁白
```
（拨弄篝火，火星像萤火虫一样飞舞）
（为大家倒热巧克力，热气腾腾）
（指向星空）"看！有流星！"
（在愿望板上挂上新的愿望卡）
（裹紧披肩，微笑看着大家）
```

## 🎪 团建活动设计

### 活动流程
1. **到达与欢迎**：大家陆续到达营地，篝火守护者欢迎
2. **Avatar展示**：每个人介绍自己的新年Avatar
3. **愿望分享**：围绕篝火分享2026年愿望
4. **虚拟体验**：音乐、美食、烟花等文字体验
5. **自由交流**：随意聊天，享受篝火时光
6. **愿望悬挂**：把愿望卡片挂到愿望板上

### 虚拟体验元素
- **音乐**：篝火边的吉他弹唱，歌曲描述
- **美食**：热巧克力、烤棉花糖、热苹果酒的描述
- **烟花**：夜空中文字描述的虚拟烟花表演
- **星空**：流星、星座、银河的文字描绘

### 互动方式
- 围绕篝火坐成一圈，轮流发言
- 通过旁白描述环境和动作
- 使用愿望卡片作为实体互动元素
- 通过星空观察引入哲学讨论

## 🎯 场景使用指南

### 何时选择这个场景？
- 需要**庆祝、分享、放松**时
- 进行**非工作性的团队建设**时
- 分享**个人愿望和情感**时
- 体验**虚拟世界的娱乐元素**时
- 讨论**存在、意义、价值**等抽象话题时

### 如何启动团建？
1. **场景设定**：描述山顶营地、篝火、星空、寒冷中的温暖
2. **Avatar引入**：篝火守护者在拨弄篝火，准备热饮
3. **欢迎仪式**："大家来了！快过来烤火，有热巧克力"
4. **活动引导**：自然引入Avatar展示和愿望分享

### 氛围维持技巧
- 通过拨弄篝火、添加木柴维持视觉焦点
- 通过倒热饮、分享食物创造温暖感
- 通过指向星空、观察流星引入宏大话题
- 通过愿望卡片创造实体互动感
